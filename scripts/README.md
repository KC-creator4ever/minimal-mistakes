# Photo Organization Scripts

This directory contains scripts for organizing and managing photos in the minimal-mistakes site.

## Scripts

- **generate_thumbnails.py**: Generate thumbnails and create/update album indices
- **assign_categories.py**: Assign photos to categories based on a mapping file
- **mapping.yaml**: YAML file mapping photo filenames to categories
